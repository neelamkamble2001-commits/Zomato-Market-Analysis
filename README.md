# 📊 Zomato Global Market Analysis Dashboard

## **Project Overview**
This project is a comprehensive **Business Intelligence Dashboard** built in Tableau. It analyzes Zomato's global footprint, focusing on the relationship between market scale, customer satisfaction, and digital adoption.

## **🚀 Key Performance Indicators (KPIs)**
* **Total Restaurants:** 9,551 (Global Inventory)
* **Average Rating:** 3.08 (Quality Benchmark)
* **Online Delivery %:** 25.66% (Digital Maturity)
* **Average Order Value:** $1,313.03 (Market Positioning)

## **💡 Key Insights**
* **Market Concentration:** India remains the volume anchor, but international hubs show high potential for premium tier growth.
* **Digital Gap:** While Asian cuisines lead in volume, Middle Eastern cuisines show the highest rate of online delivery adoption.
* **Price Dynamics:** There is a clear inverse relationship between price point and restaurant count, identifying the "Mass Market" as the core business driver.
